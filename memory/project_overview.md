# Quiz App Project

A React/Redux author-book quiz game where players match author portrait images to their books. Users can add custom authors via a form at `/add`.

## Current State (as of 2026-04-14)
- Active development, version 0.1.0
- Clean git history on `master` branch
- Recently added: form validation in `AddAuthorForm.js` and dependency updates

## Key Facts
- 6 seed authors: Mark Twain, Joseph Conrad, J.K. Rowling, Stephen King, Charles Dickens, William Shakespeare
- Game length: 10 correct answers to win
- Redux DevTools integrated for debugging
